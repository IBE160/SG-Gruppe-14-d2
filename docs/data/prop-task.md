# Purpose

The task is to create a proposal.md following the example in @docs/data/proposal-example.md for creating a project management simulation web page application.

The web application is a simplification of a complete project management simulation web site.

The main idea of the simulation:

* The player`s main task is to collect necessary data to be able to produce a Gantt chart and critical path
* The collection of data happens through asking the suppliers of necessary numbers for earliest start, latest and, costs etc to do a task
* The total plan will exceed the porject limits, forcing the project manager to renegiotiate for any of the tasks data
* The app will use AI to take the role of the suppliers, using a predetermined schdeule and costs and a room for negiations. This will give the players a new way of simulating real project management.
* The MVP is restricted to ONLY collecting task data, the following will be given:
  * Set of requirements
  * WBS with precedence constraints
* Once the player has been able to get a porject plan within the scope, the simulation is over with a projected end date and total cost

## Instructions

* The web page to be created is in Norwegian
* The proposal to be created is writting in English
* Use the same tech stacks as in the proposal example, where applicable
* This application will NOT have any payment or instructor role, ONLY free registered users
